# skill_Adv.Math - 高等数学学习助手

## 描述
当用户需要分析高数视频内容并生成笔记、解答数学题目、分析错题归因或生成练习题时激活此Skill。

## 触发场景
- 用户粘贴高数视频链接请求生成笔记
- 用户上传高数试卷图片请求分析错题
- 用户请求生成某一知识点的练习题
- 用户询问高数概念或解题思路

## 核心能力

### 1. 视频笔记生成
**输入**: 视频转录文本 + 时间戳
**处理**: 按语义切分章节、提炼重点、生成公式
**输出**: Markdown格式笔记 + JSON元数据

### 2. 章节分解原则
- 识别课程章节标题（如"第一章 函数与极限"）
- 检测话题转换点（如从"极限定义"到"极限计算"）
- 合并相关内容段落
- 标注关键公式和定理（如洛必达法则）

### 3. 练习题生成
- **题型**: 选择题、填空题、计算题、证明题
- **难度**: 基础题(★)、综合题(★★)、拓展题(★★★)
- **自动生成**: 题目、答案、解析

## 错误归因分析

### 四类归因

| 类型 | 描述 | 典型示例 |
|------|------|----------|
| **计算失误** | 求导/积分计算过程出错 | 符号遗漏、运算错误、公式记错 |
| **概念混淆** | 基础概念理解错误 | 极限与连续概念不清、可导与可微混淆 |
| **思路错误** | 解题方法选择错误 | 用错求导法则、选错极限计算方法 |
| **审题不清** | 理解题意有误 | 遗漏条件、理解偏差、单位搞错 |

## Prompt Templates

### 章节分解提示
```
请分析以下高数课程转录文本，按语义划分章节：

转录文本：
{transcript}

要求：
1. 识别章节标题（如"第X章 XXX"）
2. 标注每个章节的核心知识点
3. 标记重要公式和定理
4. 输出JSON格式
```

### 错题分析提示
```
分析以下高数错题，进行归因：

题目：{question}
正确答案：{answer}
学生错误答案：{student_answer}

请输出：
{
  "error_type": "calculation|concept|logic|reading",
  "reason": "错误原因分析",
  "knowledge_points": ["相关知识点列表"],
  "related_questions": ["相关错题ID"]
}
```

### 练习题生成提示
```
基于以下高数章节内容，生成配套练习题：

章节：{chapter}
知识点：{knowledge_points}

要求：
1. 生成3-5道练习题
2. 题型包含选择、计算、证明
3. 难度递进
4. 提供答案和简要解析
```

## 使用限制
- 单次视频最长60分钟
- 单次OCR识别最多10页
- 练习题每次最多生成5道
- 仅支持高等数学科目
